## How to use?

Berikut cara menggunakannya :
1. Clone repository ini. 
   ```
   git clone https://github.com/gabrielhtg/my-vim.git
   ```
2. Copy file `.vimrc` ke home directory kalian.
3. Selamat menggunakan.
